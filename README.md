Proyek ini adalah implementasi landing page statis untuk **Universitas Amikom Yogyakarta** sebagai bagian dari tugas praktikum pemrograman web. Fokus utama proyek ini adalah penggunaan struktur semantik HTML5 yang tepat dan styling CSS3 yang modern.

---

## 🚀 Fitur Utama

- **Semantic HTML5**: Mengikuti standar hierarki Heading (`h1`, `h2`, `h3`) sesuai instruksi.
- **Modern UI Architecture**: Desain bersih dengan tipografi yang dioptimasi dan efek bayangan (*shadow*) halus.
- **Responsive Layout**: Tampilan yang rapi di berbagai ukuran layar.
- **Interactive Elements**: Efek transisi pada gambar (hover zoom) dan link yang interaktif.
- **Clean Footer**: Identitas pengembang yang diletakkan secara elegan di bagian bawah halaman.

---

## 🛠️ Tech Stack

- **Markup:** HTML5
- **Styling:** CSS3 (Internal Stylesheet)
- **Typography:** Segoe UI / Sans-serif
